Install

Run

myskill --help

USAGE

常用命令

创建 Skill

在 ~/.codly/skills 下创建一个新 Skill 目录

myskill create commit-helper

列出 Skills

查看当前机器已安装的 Skill

myskill list

在 codly 中使用

进入 codly 后输入 /skill 选择，或 /skill-<name> 直接激活

/skill-commit-helper

免责声明

请在使用前仔细阅读

⚠️ 安装脚本

curl | bash 注意事项：安装脚本会：

• 根据当前平台从 http://codly.jiyingda.com 下载 myskill 二进制
• 安装到 ~/.local/bin/myskill（或 --system 时 /usr/local/bin）
• macOS 上去除 com.apple.quarantine 标记

请在执行前审查脚本内容。

⚠️ Skill 来源

第三方 Skill 的风险：Skill 的 SKILL.md 会作为提示注入到 codly 的 system prompt：

• 只安装来源可信的 Skill
• 在激活前阅读 SKILL.md 的 Instructions
• 注意 Skill 可能调用 exec_bash 等工具
• 不确定时不要启用
