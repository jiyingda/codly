⚠️ 安全警告

Shell 命令执行：Codly 可通过 exec_bash 执行 bash 命令。 该功能可能运行会对系统造成风险的命令：

• 修改或删除系统文件
• 更改系统配置
• 安装或卸载软件包
• 访问敏感数据

请务必在批准前审查建议的命令。 使用风险自负。

⚠️ API Key 安全

保护您的凭证：API Key 存储在 ~/.codly/settings.json 中。 请确保：

• 切勿将此文件提交到版本控制系统
• 将 ~/.codly/ 添加到全局 .gitignore
• 保密您的 API Key
• 定期检查 API 使用情况

⚠️ AI 生成代码

需要代码审查：Codly 使用 AI 生成代码和建议。 请始终：

• 在使用前审查生成的代码
• 在安全环境中充分测试
• 未经审计不要将 AI 生成代码用于关键系统
• 注意潜在的安全漏洞
